机制

引言​

“卝城”是一个人内心之缩影的实体形式。每个人都有属于自己的故事，因此也就拥有属于自己的卝城。卝城中存放着你的过往、你的愿望，以及一切纷杂的情感和经历。在这个故事里，我们暂且假定这些回忆或期许是美好的（至少一部分是），以至于你会流连其中。

人刚刚来到世界时，他的内心犹如一张白纸；于此同时，世界上多了一座全新的卝城。孩子渐渐长大，接触到世界上各样的人、事、物，来自外部的东西逐渐填满了他的内心：缤纷的生活、璀璨的文化、便捷的科技...这些东西可能会让他忘记原本的自己，最终迷失在历史的浪潮里。

但他也曾有过美好的回忆啊！那些昙花一现般的温暖、纯粹的情感、绚烂的梦想，甚至是一个模糊的模样、依稀的声音，难道就让它们就此消散，好像从未存在过一般吗？于是内心开始留恋，却又难以取舍：放弃追寻大千世界只剩下孤单的自己，亦或是承认自己的渺小随时代而去？这是《旧卝城》原本的命题。

有了这样的前提，尝试理解本文的世界观就变得稍微容易了一些。好了，接下来我们 相互道个别 或 请带着好奇来看看本文的设定 吧！

.1 世界运行​

.1.1 世界观​

（1）世界​

“这个世界只有一块近似圆形的大陆，陆地以外是无尽的大海，向海之尽头出发会到达世界的另一端，连接世界两端那道模糊的界限被称为世界的边缘。”——正文第1节

所有的故事都发生在这块陆地和这片海域上，但这个世界并不是完全封闭的，在它之外还有另一个世界（aka. 真实的世界），世界之门连接着这两个相互隔离的存在。《旧卝城》并不试图讲述世界之门以外的事情，那应该被划归为《新卝城》的范畴。

[设定说明]

两个世界各自独立，但世界之门为其提供了通信的可能，不知道这种想法更像“管道”还是“套接字”？

世界边缘的存在致使整个世界看起来像是一个球，因为无论往哪个方向直行都会回到原地。但世界的边缘确是存在的，从边缘某处的缝隙可窥见世界之门。就好像一道具有特定角度的空间裂缝一般。

（2）环境​

卝城的陆地其实并不很大，以真实世界作为对比，它的大小甚至顶不上一个中国的面积，这是因为对于大多数人而言，他的一生能够涉足的土地恐怕也不会超过如此大小。在陆地上存在着（过）为数不多的城市，城市周围都有附属的原野或者乡村，除此之外世界各处还散落着众多无名的聚居区。

卝城存在四季变换，四季不分区域且具有明显的时间间隔，因此卝城世界的植被和生物更类似于北方地区，其中当然也可以包括真实世界中不存在之物。

卝城的地理环境种类众多，包括山地、平原、盆地等，但是没有沙漠、冰原和非常高的山峰。

（3）科技​

卝城世界的科技水平存在两极分化，“外来”的人掌握远超现代的知识和技术，而“原住民”与这个世界的亲和性更好一些。然而从武力的角度来说，科技发达的一方却并不一定强过本地的一方，这很容易理解：你拥有对外来信息进行选择的自由，因此相异武力体系共存的设定并不显得矛盾。

特别地，故事中的科技相比于真实世界在某些方面的水平和认知无法完全匹配，如空中交通建设、天文地理等，这是卝城世界的特殊性决定的。

（4）资源​

卝城世界的资源来自各种各样的矿石，它们中的一些能够提供能量或者作为建筑材料，还有一些则可以保存信息或者“与世界交互”（详见 <设定集-物品-矿石类> 章节）。随着对矿石资源的开发和利用，外来者逐渐窥探到这个世界的运行规律，也凭借这种方式在卝城的世界扎下根来。

[设定来源]

在真实的世界中，我们通过与他人交谈完成沟通和互动，通过聆听对方的故事来联系彼此之间的感情。矿石就是记忆的碎片，由此可以理解为什么卝城世界的矿石被设定为可以存储信息或者完成连接。

[设定扩展]

在真实的世界中，将内心完全暴露给别人是危险的行为，因为那是我们最脆弱的东西。在卝城的世界中同样如此，外来者会借助矿石利用或破坏卝城。

（5）经济​

经济体系是构建一个社会必不可少的要素，卝城的世界中同时存在两种经济体系，它们分别是：与真实世界中“人民币”等值计量的“元”和以便于分配为目的的“积分”。前者适用于与外来者交易的场景，后者适用于利燕组织的管辖范围。

（6）时间​

在卝城世界，主故事线的时间的流速与真实世界基本相同，只是在卝城世界建立初期，时间却流逝得要慢一些，慢到足以容纳世界演变至今的一切变化。前文所述，卝城是一个人的内心世界，因此这个世界的发展方向基本与人的一生相同（即建立->发展->没落->（尝试重建）->结束，对应人的出生->成长->老去->死亡）。

卝城世界的时间也遵循日出日落，一天24小时，这些自然时令的变化均为卝城主动控制形成，详见 .1.3 控制中心 章节。

（7）方向​

在真实的世界中，东南西北根据地球的磁场进行区分，但卝城的世界是一个“平面”，地磁场的概念仅用于支撑长尔来持有的科技运作。卝城世界中的东南西北是根据天上四星所在方位进行区分的。四星两两相对，可相连为一个正方形，光芒分别为红、蓝、橙、绿。卝城世界的人们以红星方向为北，橙星方向为南，蓝星方向为东，绿星方向为西。

（8）语言​

故事通篇使用中文和英文，但卝城世界中没有国别，因此并不严格对语言做区分。默认情况下场景中涉及的信息/提示/文字/说明等内容均基于中文叙述。

[设定说明]

语言的本质只是信息的不同编码方式。如果在架空世界中设定字母“A”是中文里的特殊汉字，拼音为“ei”，想来也只是稍微怪异而已。

（9）单位和基准度量衡​

真实世界中的基本单位的定义在卝城世界中被直接引用。含义相同，但不讨论来源出处。

[设定说明]

历史的传承得以将千年以来人类对世界的探索成果流传给后世，但对于大多数人而言，他们并不一定需要去了解那段历史。例如当你听见钟表滴答感到时间流逝，却大可不必去理会那一秒的跨度究竟有多长。

.1.2 主要地点​

本节所述“地点”为在通常情况下以实体形式存在的场所或环境，或者是普通人认知中对“特定地理位置”的代称。

（1）特里克斯​

特里克斯城位于整个世界平面的中心。特里克斯主城范围约78平方千米，周围还包含一些附属的地点，它们分别是：

  1. 位于主城东南方向的 乡村 ；
  2. 位于主城正南方向的 独山亭 ；
  3. 位于主城正北方向的 风廊 ；
  4. 位于主城西偏北方向的 平原 。

[编写设定]

在本文中有一个暗示：一切偏向/归属于卝城一方的势力均称特里克斯城市为“特城”，而所有敌对/路人势力均以全名进行称呼。

特里克斯由利燕组织依照欣羽提供的蓝图建造和管理，管理中心由原赫拉纳居民（叶姓家族）代工。特里克斯具有 恢复机制 ，在受到大面积破坏时可以从时间线的备份中还原到破坏前的状态，该机制依赖于琉璃长廊。

在特里克斯管理中心任职的五大管理拥有特城赋予他们的 “职能” ，职能的作用是在特里克斯所属范围内保护持有者免受物理伤害。

（2）长尔来​

长尔来城市是外来者们最大的聚居区，位于整个世界平面的西北侧。长尔来势力的影响范围几乎横跨半个世界。

紫溪 和 明珠 是长尔来的两大科技，前者以武器研发为主，后者以防护系统研发为主，两者相互牵制。除此之外，还有众多不可忽视的小组织或团体等，它们依附于两大科技，共同形成两大势力。

在紫溪和明珠之外，还存在长尔来的领导群体，该群体活动于幕后不为大众所知，但却影响甚至操控着两大势力的决策和行动。

[编写设定]

长尔来的人从 世界之门 进入到卝城的世界，并始终维持着与外界的联系，但这种联系是隐藏的、被掩盖的，因此在《旧卝城》故事中，不会刻意强调甚至提及有关世界之门的事情。

有关世界之门的设定详见 .3.2 世界之门 章节。

（3）赫拉纳​

叶姓家族擅长建造各种建筑，赫拉纳为曾经的叶姓家族聚居区。

部分叶家人具备天生对某些矿石的亲和性，可以辅助建筑的建造过程，因此被称为“构造者”。

[设定说明]

“赫拉纳城代表了一个人的成果或心血，外界对其有兴趣想要了解（目前不清楚是善意或恶意），结果触及到了它脆弱的一面导致其覆灭。这是一件值得痛心的事情，但曾经存在过的美好却依然会被记住，成为这个人宝贵的回忆（赫拉纳的一部分被并入特里克斯）。而这份创造的源泉只要存在，美好就不会枯竭（仅剩的叶家人丁建造了管理中心）。”——正文第6节-主旨

（4）西南山林​

西南山林是漈组织的所在地，虽然漈的管理通常分散在世界各处，但只要前往西南山林，却总能找到他们。

.1.3 控制中心​

在卝城的世界中，世界的事物运行规律均由控制中心决定，时间流逝、日出日落、季节变换...控制中心好像人体中的低级神经中枢一般，不受两大组织的控制，自然而然、周而复始地运作着。

控制中心没有确切的地理位置，但可通过 石台地堡入口进行访问，访问操作必须有 利燕 或 漈 组织最高领导者的授权才能执行。访问时，控制中心主动实体化，为访问者提供一个与卝城世界隔离的空间。实体化空间表现为一个温馨的小房间，夕阳的余晖洒在窗旁的小书桌上。

实体化的控制中心提供以下操作接口：

  1. 房门和窗户为退出点，供访问者离开；
  2. 房内书架上的书籍和书桌上的笔记本等可作为指令执行的输入媒介，通过修改或增删其上的内容可操控世界上某处的运行状态；
  3. 通过查阅书本上的信息可访问控制中心的日志——世界的历史记录。

实体化的控制中心具有以下特性：

  1. 房间内的夕阳是不会落下的，始终处于黄昏的状态；
  2. 房间内的时间相对于外界是静止的，访问者退出时将回到进入时的时间点；
  3. 房间内的环境是不可破坏的，每一次对控制中心的访问均会产生不同的实体房间；每一次离开房间，该房间都会被销毁，因此无法通过房间破坏控制中心本身；
  4. 房间内的物品是有访问权限的，根据访问者的权限不同，可从书本上查阅或增删的信息也不同。

控制中心的真正实体即欣羽本身，这意味着控制中心本身是具有意识的——它的意识即欣羽的意识。（这可能会成为一个大坑）

控制中心在实体化房间以外也提供了一种对世界运行的访问和修改的能力，详见 .5 权限 章节。

.2 备份和恢复​

记忆是一个人对过往经历的记录，只是它是残缺的，因此不可复原。对于一个世界而言，它的记忆便是对其间所存全部事物的发展变化历程。

卝城的世界具有三种各不相同的备份机制。

.2.1 特里兰​

特里兰存放了整个世界最开始样子的缩影，是世界创建伊始就固定好的。特里兰与长尔来共享一个地理位置，但却存在于不同的空间中。

从外部进入特里兰空间时，需要露泠石（参见 <设定集-物品-矿石类> 章节）作为启动燃料。故事中，特里兰因为部分关键事物的丢失和变更而失效，整个空间被废弃。

.2.2 琉璃长廊​

琉璃长廊和控制中心一样，也没有确切的地理位置，但可通过 石台地堡入口进行访问，访问操作必须有 利燕 或 漈 组织最高领导者的授权才能执行。

琉璃长廊中存放了 特里克斯 的全部历史时刻，是特城恢复机制的支撑。琉璃长廊是一条两端 无限延伸 的通道，一端通向过去，一端通向未来。通道四围有流光镌刻的壁画，其中记录着过往发生的事情。

[设定说明]

这个世界虽然有始有终，对应着一个人的出生和死亡，然而当这个人活着的时候，他既无法清晰地追忆出生的时刻，也无法准确地触碰到死亡的瞬间。琉璃长廊与此类似，但站在旁观者的角度，一个人的人生不过是一个片段，线段的头尾清晰可分。

琉璃长廊的每次恢复都以一定程度上的损耗作为代价，损耗表现为壁画的流光逐渐熄灭或消失。故事中，琉璃长廊最终因恢复次数超出承载能力而失效。

[设定说明]

当仅仅试图通过回忆过往的美好安抚受伤的心灵时，随着受伤次数的增加，回忆能够带来的安慰逐渐消逝。

琉璃长廊和特里兰不属于广义上的“地点”，它们只是镜像存储的实体化形式。

.2.3 欣羽​

欣羽是与世界相伴而生的。它不仅记录了世界开始的样子，还记录了 世界 发展至今所经历的变迁，因此可以这样说：欣羽就是这整个世界。

欣羽和琉璃长廊、特里兰有相似之处，但它并不仅仅是一个“存档”，而是真正有意识的。欣羽虽然可以充当世界的备份，但从欣羽所还原出来的世界只能是一个“镜像”的世界，而不是“全新”的世界。这导致在还原之后，原来的世界上无论发生了什么，其造成的影响都将被保留。

以欣羽作为源重置世界的操作必须由 守护者 完成（守护者是由欣羽挑选并信任的人），这不会导致欣羽死亡或消失，而是会生成一个只有欣羽和它的守护者存在的新世界。至于原来的世界……也许会消失、或成为平行世界、或成为和特里兰一样的存在吧，设定中将其作为悬念不予定义。

欣羽在故事中的实体化形象为一只猫形生物，按照它自己的说法“习性与猫差不多”，但不具备正常猫物种拥有的一些特别结构如舌上倒刺、原始袋、肛门腺或生殖系统等，因此虽外形偏向雌性，但没有明显的性格倾向，更不会发情……

.3 出入口​

出入口指的是进出特殊地点或访问特殊事物的途径或方法。

.3.1 石台地堡​

石台地堡可连接到琉璃长廊实体，由利燕和漈共同看守。石台地堡位于整个世界的最南方，但只有越过最北方矿山的界限后才能看到，直接往正南走是无法到达该位置的，路线充当了访问此处的“密码”。

.3.2 世界之门​

世界之门用以指代一切连接矿城内部和真实世界的途径，其实体化的表现包括但不限于空间裂缝、时空隧道等方式，在《旧卝城》故事中，不对世界之门做特别具体的阐述。

.4 传送点​

传送点特指由利燕组织在特里克斯及其所属范围内设立的7个传送点，它们分别是：风廊、管理中心、桃山、城南小站、乡村、东南站和矿山。传送点的设立方便了管理们展开活动，也为特里克斯城际间物资运输等提供了基础设施支持。

不同的传送点有不同的启动方式，总的来说包括以下几种方法：

  1. 密码路径

    与石台地堡类似，访问者在一段时间内的行走路线充当访问传送点的密码。

  2. 访问许可

    已持有访问许可的人可为新人向看守申请新的许可，访问许可保证了持有者对至少2个传送点的使用权。权限持有者凭借权限即可使用传送点，无需特别的使用许可。

  3. 特殊的映射连接

    权限持有者可凭借权限直接将物品传送到指定的传送点，传送点看守将获知物品来源并代为保管。

通常情况下，传送点的看守并非是真实存在的人，而是世界控制中心部分运行逻辑的映射，但当管理中心为传送点指派看守时，指派的看守将替代原本的映射而存在。

.5 权限​

卝城的世界存在一种机制，可以切入事物的运行规律，直接控制它们运行的结果，这种机制被称为“权限”。权限由控制中心提供支持且直接作用于控制中心本身，以调用对指定事物的操控方法，控制中心会对权限的每次使用执行审核。权限的调用过程一般具有时效性，这受到使用者本身意识的影响，因为权限本质上只是提供了一种调用，真正的控制操作是使用者本人执行的。

权限可以被继承、共享和叠加。
